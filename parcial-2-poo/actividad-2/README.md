Alumno: Mario Alejandro Hernandez Lopez

Objetivo de la practica: Crear un sistema de ingreso de usuarios utilizando transacciones con promesas, commits y rollback con errores simulados con PDO + MySQL

Requisitos: PHP 8, Servidor Apache XAMPP, Servidor SQL XAMPP, PHPMyAdmin

Ruta del ejercicio: http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-2-poo/actividad-2/index.php

Instrucciones de ejecucion: 

1. Clona el repositorio de la actividad 1 en la carpeta htdocs en la carpeta de XAMPP
2. Activa los servicios APACHE y SQL de XAMPP
3. Entra a localhost/phpmyadmin en tu navegador e importa la base de datos de la actividad del archivo phppdobd.sql
4. Introduce la ruta de ejecucion antes mostrada en el navegador para entrar a la practica

Repositorio Github de la practica: https://github.com/OurScorpion045/desarrollo-web-avanzado-fimaz-uas/tree/main/parcial-2-poo/actividad-2
